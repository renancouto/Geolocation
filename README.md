#Geolocation
[Geolocation](http://jsbin.com/avudev) is a javascript that detects user's location and returns:

* Country Code
* Country Name
* City
* Region
* Region Name
* Latitude
* Longitude
* Postal Code
* Area Code
* Metro Code

Geolocation depends on j.maxmind.com/app/geoip.js, and a demo can be found [here](http://jsbin.com/avudev).
